# AntiCloud / Tierra Firme - Explicación del Proyecto

## ¿De qué trata esta aplicación?

**AntiCloud** (también llamado "Tierra Firme" en la interfaz) es una **pizarra digital interactiva** diseñada principalmente para **músicos, DJs y creadores de contenido** que necesitan organizar visualmente elementos como:

- **Acordes musicales** - Para crear progresiones, setlists visuales o estudiar teoría musical
- **Cajas de texto** - Para notas, letras o cualquier contenido de texto
- **Contadores y temporizadores** - Para gestionar tiempo o puntos en sesiones live
- **Ilustraciones** - Imágenes pixel art para ambientar la pizarra

## ¿Para qué sirve?

1. **Organización visual live** - Principalmente usada en eventos, sesiones de estudio o práctica musical
2. **Planificación de sesiones** - Crear esquemas visuales de canciones o transiciones
3. **Estudio de teoría musical** - Visualizar progresiones de acordes y escalas
4. **Pizarra colaborativa** - Múltiples usuarios pueden manipular elementos en tiempo real (tbd)

## Características principales

- Múltiples tableros ("pizarras") conmutables
- Sistema de **capas** para ordenamiento visual
- **Conexiones** entre elementos (flechas entre items)
- **Inventario** para guardar y reutilizar elementos
- Temas visuales (estilos pixel/retro)
- Exportación a imagen o archivo `.anticloud`
- Modo móvil con controles táctiles

## Estado actual del proyecto

El proyecto está **bastante completo** en términos de funcionalidad core:
- Interfaz completa con React + TypeScript
- Sistema de arrastre y manipulación de elementos
- Múltiples tipos de elementos implementados
- Modales para añadir/editar elementos
- Persistencia de datos en IndexedDB
- Empaquetado para escritorio (Electron + Tauri)
